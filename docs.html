Welcome to DataLens — Quick start & help

Client-side data exploration, API testing and light ETL — fast, private, and built for developers & analysts.

Getting started

Two minute setup • No backend required

1. Open Data Explorer → Upload a dataset (CSV/XLSX/JSON).
2. Use the global search to filter rows in real-time.
3. Create quick charts or export visible rows.

Sample dataset

Use "Sample dataset" to try features instantly without uploading files.

Persist

Enable local persistence to preserve the dataset in your browser (clears on user action).

Secure by default

All processing is local. Remote APIs are only called if you make requests.

File upload & supported formats

CSV • XLSX • JSON • TXT • XML

Drag & drop or click the upload area. DataLens auto-detects encodings and column types. For Excel files, a sheet chooser will appear — the first sheet is chosen by default. 

{/* Example: programmatic import (client-side) */}

Data inspection

Preview • Types • Row & column counts

Click any column header to view summary stats (unique, nulls, sample values). Use the column menu to rename, change type, or hide a column.

Summary panel

Shows min/max, mean, top values and sample rows for the selected column.

Schemas

Generate a lightweight schema preview that can be exported as JSON schema.

Search & Filters

Global search + column filters

Global search scans visible columns. Column filters allow exact match, contains, numeric ranges, and date ranges. Combine multiple filters to refine results.

Filter types

Text: contains/equals • Number: range • Date: calendar picker

Save & reuse

Saved filters can be named and reapplied to new datasets.

Quick charts

Instant bar/line/pie visualizations

Select a categorical column and a numeric column then click Chart. Charts render client-side and can be exported as PNG.

API Studio

Inspect headers, body, latency, and preview JSON

Use the API Studio to craft requests (GET/POST), attach custom headers, send sample payloads and inspect responses. Responses can be imported into the Data Explorer for analysis.

Quick tip

Enable Save requests to reuse common test scenarios across sessions.

Export & Reset

CSV / XLSX / Original file

Export only what you see (visible columns & active filters) or download the original file. Use Reset to clear filters and revert to the raw view.

Privacy & Security

Everything runs in your browser

DataLens does not upload your data by default. LocalStorage persistence is optional and user-controlled. If you call external APIs from the API Studio, those requests go to the target servers (be mindful of sensitive payloads).

FAQ & Support

Quick answers
How do I clear persisted datasets?

Open Settings → Storage → Clear persisted datasets.

Where does my data go?

Unless you call a remote API, all data stays in browser memory/localStorage.
