Welcome to QBITRIX DataLens

Privacy & Security

Data never leaves the browser by default. LocalStorage persistence is optional and user-controlled. If you call external APIs from the API Studio, those requests go to the target servers (be mindful of sensitive payloads).

FAQ & Support

How do I clear persisted datasets?

Open Settings → Storage → Clear persisted datasets.

Where does my data go?

Unless you call a remote API, all data stays in browser memory/localStorage.
