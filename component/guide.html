DataLens — User Guide
Client-Side Data Explorer — Quick start & tips
Version 1.0.0.0 • © Qbitrix Datalens

Welcome! This guide helps you quickly understand how to use DataLens to upload, explore, analyze, and export datasets — and how to use API tools effectively.

1. File Type / Upload

Auto-detects file format. Drag & drop or click the upload area. Supported: CSV, XLSX, JSON, TXT, XML.

2. File & Data Info

View filename, mime type, size, row & column counts. Use the "Persist" toggle to keep data in localStorage.

3. Global Search

Quickly search across all visible columns — use it for fast filtering and discovery.

4. Columns & Filters

Toggle column visibility, create column-specific filters and refine the dataset view.

5. Quick Charts

Select a categorical & numeric column and draw bar/line/pie charts instantly.

6. Export & Reset

Export visible rows to CSV/XLSX, download the original file, or reset the view to clear filters.

Tips
  • Use the Sample dataset button to try features instantly.
  • Enable "Persist" to keep the dataset during development sessions.
  • Use API Studio to mock endpoints and test with API Lens.
Privacy

All data processing happens in your browser — nothing is uploaded to any server unless you explicitly call remote APIs. LocalStorage is optional and can be cleared anytime.
